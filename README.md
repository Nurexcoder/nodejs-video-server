# nodejs-video-server
## 🚀 Quick Start

1. **Install dependencies.**

   ```bash
   npm install
   ```
   ##### or
    ```bash
   yarn 
   ```
   

1. **Start project locally.**

   ```bash
   npm start
   ```
   ##### or
    ```bash
   yarn start
   ```
1. **Open the source code and start editing!**

## ℹ️ About Project
Learn to build node js video server which send video source chunk by chunk
